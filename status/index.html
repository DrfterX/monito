Features

Built for the monitoring-obsessed

One-minute check intervals, real-time status, multi-window uptime statistics, and email alerts — all running on Cloudflare's global edge network. Zero infrastructure to manage.

📡

Minute-level checks

Every monitored endpoint is checked once per minute. You get near-real-time visibility into service health.

📊

Uptime statistics

24-hour, 7-day, and 30-day uptime windows. See trends at a glance with per-endpoint and aggregate views.

🔔

Email alerts

Get notified when a service goes down. Configurable thresholds prevent alert fatigue from transient failures.

⚡

Edge-deployed

Runs on Cloudflare Workers with D1 storage. Global edge network means checks originate from the same infrastructure your users hit.

📈

Response waveforms

Visualize latency trends with oscilloscope-style waveform charts. Spot degradation before outages happen.

🔌

API-first design

Full REST API for managing monitors, querying status, and fetching uptime data. Integrate with your own tools.
